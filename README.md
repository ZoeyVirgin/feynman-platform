# Feynman Platform / Feynman 平台

## 项目简介 / Project Overview

**中文**  
Feynman 平台 —— 一个多平台应用开发框架，包含前后端基础设施，支持 Web、移动端及未来 OS 扩展，旨在提供可复用、可扩展的学习与实验环境。

**English**  
Feynman Platform — A multi-platform application framework with frontend and backend foundations, supporting Web, mobile, and future OS adaptations. Designed to provide a reusable and scalable environment for learning and experimentation.
